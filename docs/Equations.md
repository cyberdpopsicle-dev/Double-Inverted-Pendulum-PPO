# Double Inverted Pendulum Dynamics

## State Vector

The system state is defined as

[
x =
\begin{bmatrix}
\phi_1 \
\dot{\phi}_1 \
\phi_2 \
\dot{\phi}_2
\end{bmatrix}
]

where

* (\phi_1) : first link angle
* (\dot{\phi}_1) : first link angular velocity
* (\phi_2) : second link angle
* (\dot{\phi}_2) : second link angular velocity

---

## Friction Torques

Viscous damping is modeled as

[
M_{R1}=R_{p1}\dot{\phi}_1
]

[
M_{R2}=R_{p2}\left(\dot{\phi}_2-\dot{\phi}_1\right)
]

---

## Equations of Motion

The nonlinear dynamics are expressed in matrix form as

[
H(\phi)
\begin{bmatrix}
\ddot{\phi}_1 \
\ddot{\phi}_2
\end{bmatrix}
=============

\begin{bmatrix}
F_1 \
F_2
\end{bmatrix}
]

where the inertia matrix is

[
H=
\begin{bmatrix}
H_{11} & H_{12} \
H_{21} & H_{22}
\end{bmatrix}
]

with

[
H_{11}=a_{11}
]

[
H_{12}=a_{12}\cos(\phi_1-\phi_2)
]

[
H_{21}=a_{12}\cos(\phi_1-\phi_2)
]

[
H_{22}=a_{21}
]

---

## Nonlinear Generalized Forces

[
F_1=
-a_{12}\sin(\phi_1-\phi_2)\dot{\phi}*2^{,2}
-a*{13}\sin(\phi_1)
+u
-M_{R1}
]

[
F_2=
a_{12}\sin(\phi_1-\phi_2)\dot{\phi}*1^{,2}
-a*{22}\sin(\phi_2)
-M_{R2}
]

where

[
u
]

is the control torque.

---

## Forward Dynamics

The angular accelerations are obtained from

[
\begin{bmatrix}
\ddot{\phi}_1 \
\ddot{\phi}_2
\end{bmatrix}
=============

H^{-1}
\begin{bmatrix}
F_1 \
F_2
\end{bmatrix}
]

Using analytical inversion,

[
\ddot{\phi}*1=
\frac{H*{22}F_1-H_{12}F_2}
{\det(H)}
]

[
\ddot{\phi}*2=
\frac{-H*{21}F_1+H_{11}F_2}
{\det(H)}
]

where

[
\det(H)=H_{11}H_{22}-H_{12}H_{21}
]

---

## State-Space Form

The complete state derivative is

[
\dot{x}
=======

\begin{bmatrix}
\dot{\phi}_1 \
\ddot{\phi}_1 \
\dot{\phi}_2 \
\ddot{\phi}_2
\end{bmatrix}
]

---

## Numerical Integration

A fourth-order Runge–Kutta (RK4) method is used for state propagation.

[
k_1=f(x_n)
]

[
k_2=f!\left(x_n+\frac{h}{2}k_1\right)
]

[
k_3=f!\left(x_n+\frac{h}{2}k_2\right)
]

[
k_4=f(x_n+hk_3)
]

[
x_{n+1}
=======

x_n
+
\frac{h}{6}
\left(
k_1+2k_2+2k_3+k_4
\right)
]

with

[
h=0.002\ \text{s}
]

---

## Control Input

The control torque is generated by a PPO policy

[
u=\pi_{\theta}(x)
]

with

[
x=
\begin{bmatrix}
\phi_1 &
\dot{\phi}_1 &
\phi_2 &
\dot{\phi}_2
\end{bmatrix}^{T}
]

and

[
u\in[-6,6]\ \text{Nm}
]

The target equilibrium is

[
(\phi_1,\phi_2)=(\pi,\pi)
]

corresponding to the upright unstable configuration.
